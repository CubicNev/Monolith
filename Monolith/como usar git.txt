INICIOS A GITHUB

1.- Vas al directorio donde esta el proyecto
je C:\Usuarios\Documentos\Proyectos\Monolith

2.- Copeas el proyecto a tu compu:
	git clone https://github.com/CubicNev/Monolith.git

3.- Cuando alguien haga cambios al proyecto tu haces:
	git pull
    Con esto traes la version mas actual a tu carpeta, basicamente la actualizas

4.- Subir tu version
	hacer pull
	git add . 
	git commit -m "Agregue modulo Registro"
	git push
